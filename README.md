# Student Management System(Java)

This project is a console application writen with the Java programming language, that tracks student information.
The user can view grades(freshmen, softmore, etc), add and remove students, update student information, view reports
on the whole school or individual grades, and more. 

# Features

- Create and delete students from the system
- Change, add, and delete classes
- View grades(freshmen, softmore, juniors, seniors) displaying all of their GPA's, honors and valedictorian status.
- Update student information - add classes to schedule, change class grades
- View reports - shows the valedictorian, honors students, average class or school GPA, and number of students in that grade
